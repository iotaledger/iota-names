# Subnames

Subnames are names you create under a name you already own. They help you organize identities under a main name and can be shared with other users.

## Create a subname

1. Under **Your Names**, find the name you want to create the subname under.
1. Click **+ Add Subname** to open the dialog and enter the subname.
1. Choose whether to allow the subname to create its own subnames and to renew its expiration.
1. Find the new subname listed under **Your Names**.

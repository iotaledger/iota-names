# IOTA-Names User Documentation

Build your on-chain identity in web3 with IOTA-Names. Using IOTA-Names, you can register a name (any name that isn't already registered) and use it for on-chain transactions. 

You can use the **IOTA-Names Dapp** link in the top nav bar to jump over to IOTA-Names and register some names, or configure the ones you already own.

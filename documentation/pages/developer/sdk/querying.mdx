# Querying

Use the `IotaNamesClient` that was previously created to query on-chain data.

## Querying a NameRecord

You can query a name record's latest state by calling the `getNameRecord` method on the `IotaNamesClient` instance.

```js
const nameRecord = await iotaNamesClient.getNameRecord('demo.iota');
console.log(nameRecord);

// Example output
// {
//   name: 'demo.iota',
//   nftId: '0xMyNftId',
//   targetAddress: '0xTargetAddress',
//   expirationTimestampMs: '1746018571863',
//   data: {
//      avatar: '0xAvatarObjectId',
//      contentHash: 'content',
//   },
//   avatar: '0xAvatarObjectId',
//   contentHash: 'content',
// }
```

## Query active pricing

Call the `getPricelist` method on the `IotaNamesClient` instance to query the active names price list.

```js
const priceList = await iotaNamesClient.getPriceList();
console.log(priceList);

// ([name_length_from, name_length_to]: price. Prices are in NANOS; 1 IOTA = 1_000_000_000 NANOS)
// Example output:
// {
// 	[ 3, 3 ] => 1500000000000,
// 	[ 4, 4 ] => 250000000000,
// 	[ 5, 63 ] => 50000000000
// }
```

## Query active renewals pricing

Call the `getRenewalPricelist` method on the `IotaNamesClient` instance to query the active renewals price list.

```js
const renewalPriceList = await iotaNamesClient.getRenewalPricelist();
console.log(renewalPriceList);

// ([name_length_from, name_length_to]: price. Prices are in NANOS; 1 IOTA = 1_000_000_000 NANOS)
// Example output:
// {
// 	[ 3, 3 ] => 500000000000,
// 	[ 4, 4 ] => 250000000000,
// 	[ 5, 63 ] => 50000000000
// }
```

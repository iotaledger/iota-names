# IOTA-Names Documentation

IOTA-Names documentation is divided into two main sections: [**User**](./user.mdx) and [**Developer**](./developer.mdx). The **Developer** section provides integration details for adding IOTA-Names to your projects. The **User** section provides instructions on how to use the IOTA-Names portal to register names and integrate them with your web3 identity.

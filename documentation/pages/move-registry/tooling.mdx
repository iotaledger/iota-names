# Tooling

Two main tools are available for use with the Move Registry (MVR):

- [A plugin](./tooling/typescript-sdk.mdx) for the IOTA TypeScript SDK that enables building programmable transaction blocks (PTBs) using MVR names.
- An [MVR CLI tool](./tooling/mvr-cli.mdx) for Move development.

<Callout type="info" emoji="">
To learn more about the SDK plugin API, see [Plugins](https://sdk.iotaledger.com/typescript/plugins) in the IOTA TypeScript SDK documentation.
</Callout>
